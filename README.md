# AWS Summit Paris 2026 - DAT309 : Un guide pratique des fondations de données pour l'IA agentique

Bonjour,

Merci d'avoir assisté à notre session **DAT309 — Un guide pratique des fondations de données pour l'IA agentique** lors de l'AWS Summit Paris 2026.

Nous espérons que les patterns et concepts abordés — Model Context Protocol (MCP), Retrieval Augmented Generation (RAG), gouvernance des données, et l'intégration avec Amazon Bedrock AgentCore — vous seront utiles dans vos projets.
